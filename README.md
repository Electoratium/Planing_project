# How to run project #

## Requirements ##

python3, pip3 nodejs

## Frontend ##
    1. cd frontend
    2. npm install
    3. npm start
## Backend ##
    1. Open second console
    2. cd backend
    3. pip3 install -r requirements.txt
    4. cd app && python3 managers.py migrate
    5. python3 manage.py runserver
